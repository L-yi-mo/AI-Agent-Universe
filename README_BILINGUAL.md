## 🚀 快速开始 | Quick Start

### 基础使用 | Basic Usage

#### 🇨🇳

```python
from ai_agent_universe import Agent

# 创建 Agent
agent = Agent(
    role="助手",
    goal="帮助用户完成任务",
    model="gpt-4",
    verbose=True
)

# 执行任务
response = agent.execute("分析当前目录下的所有 Python 文件")
print(response)
```

#### 🇺

```python
from ai_agent_universe import Agent

# Create Agent
agent = Agent(
    role="Assistant",
    goal="Help users complete tasks",
    model="gpt-4",
    verbose=True
)

# Execute task
response = agent.execute("Analyze all Python files in current directory")
print(response)
```

### 多 Agent 协作 | Multi-Agent Collaboration

#### 🇨🇳

```python
from ai_agent_universe import Agent, Crew, Task

# 定义 Agent
researcher = Agent(role="研究员", goal="信息收集")
analyst = Agent(role="分析师", goal="数据分析")
writer = Agent(role="作家", goal="报告撰写")

# 创建任务
tasks = [
    Task(description="收集最新 AI 资讯", agent=researcher),
    Task(description="分析收集的信息", agent=analyst),
    Task(description="撰写分析报告", agent=writer)
]

# 执行
crew = Crew(agents=[researcher, analyst, writer], tasks=tasks)
result = crew.run()
```

#### 🇺

```python
from ai_agent_universe import Agent, Crew, Task

# Define Agents
researcher = Agent(role="Researcher", goal="Information gathering")
analyst = Agent(role="Analyst", goal="Data analysis")
writer = Agent(role="Writer", goal="Report writing")

# Create tasks
tasks = [
    Task(description="Collect latest AI news", agent=researcher),
    Task(description="Analyze collected information", agent=analyst),
    Task(description="Write analysis report", agent=writer)
]

# Execute
crew = Crew(agents=[researcher, analyst, writer], tasks=tasks)
result = crew.run()
```

## 🔧 配置选项 | Configuration Options

### LLM 配置 | LLM Configuration

```yaml
llm:
  providers:
    openai:
      api_key: ${OPENAI_API_KEY}
      models: ["gpt-4", "gpt-3.5-turbo"]
    anthropic:
      api_key: ${ANTHROPIC_API_KEY}
      models: ["claude-3-opus", "claude-3-sonnet"]
    local:
      api_base: "http://localhost:1234/v1"
      models: ["local-model"]
  
  default_provider: openai
  default_model: gpt-4
```

### 安全配置 | Security Configuration

```yaml
security:
  # 代码执行前需要确认 | Require confirmation before code execution
  require_code_approval: true
  
  # 允许的命令 | Allowed commands
  allowed_commands:
    - "ls"
    - "cat"
    - "python"
  
  # 禁止的路径 | Blocked paths
  blocked_paths:
    - "/etc"
    - "/system32"
  
  # 沙箱模式 | Sandbox mode
  sandbox_enabled: true
```

## 🔒 安全说明 | Security Notice

### ⚠️ 重要安全提示 | Important Security Notice

#### 🇨🇳

**代码执行风险**：Agent 执行的代码可能修改文件系统或系统设置

**建议操作**：
- 在生产环境使用前，先在隔离环境测试
- 启用沙箱模式
- 配置适当的权限限制
- 定期备份重要数据
- 启用完整的日志记录，定期审查 Agent 行为

#### 🇺

**Code Execution Risk**: Code executed by agents may modify file systems or system settings

**Recommended Actions**:
- Test in isolated environment before production use
- Enable sandbox mode
- Configure appropriate permission restrictions
- Regularly backup important data
- Enable complete logging and regularly review agent behavior
